## How to use

> Open the terminal in the folder you want to launch

> Clone the repo
```bash
git clone https://github.com/IweZix/BIN2-Q1-Javascript
```

> Install dependencies
```bash
npm install
```

> Start the server
```bash
npm start
```

> Open the browser
```bash
http://localhost:3000
```
